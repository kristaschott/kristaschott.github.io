A Review of State Websites

Alaska

  • I like the colors of the website.
  • The menu bar can be difficult to navigate because if you don't hover over it just right, the drop down options won't show up.
  • The website does not work in multiple screen sizes.
  • The website has a pretty basic design. There is nothing too exciting about it.

Iowa

  • A fun interactive website. It is very eye-pleasing.
  • The menu is pretty easy to navigate. I like how when you click on the menu icons it offers a drop down box with more menu items.
  • It is pretty easy to find everything you’re looking for a they have a really interactive search box at the top of the page.
  • The website works well in multiple screens.

Oklahoma

  • The website is somewhat boring looking, but it does have a lot of pictures.
  • There is no menu to navigate. I don’t like that.
  • Because of the lack of menu, it is hard to find what you’re looking for.
  • The website does work well in multiple screen sizes, though.

Tennessee

  • Good interactive menu at the top of the page. I also like how they have pictures for each of the menu items lower down on the page.
  • The colors are kind of boring.
  • The menu is easy to navigate and things load quickly.
  • It’s pretty easy to find what you’re looking for on the page.
  • The website does work well in multiple screen sizes.

Vermont

  • I love all the pictures they use on their home page. It makes it a very interactive experience.
  • The website loads quickly.
  • They have a nice menu with explanation for each of the items in the menu which I really like.
  • There aren’t a lot of colors, but that’s almost a good thing with all the pictures on the page.
  • Information is easy to find through the menu.
  • The website works in multiple screen sizes.

Ranking of the websites

  1. Vermont
  2. Iowa
  3. Tennesee
  4. Oklahoma
  5. Alaska

Overall, all of these websites loaded quickly for me, so that was not a factor used to rate them. Some websites were a little more exciting than others. Vermont and Iowa were a close first and second. Ultimately, I chose Vermont as #1 because of the pretty pictures used on the home page and their interactive menu. I really liked how they describe each menu item in the drop-down box. It makes it very easy to find where you need to go. Tennessee and Oklahoma have very similar websites. They are cool and have good features to them, but just aren’t quite as pleasing to the eye. Alaska comes in last because of their design and layout of the website. To be honest, it is just pretty boring. There is nothing about it that pops out to me and the website does not work in multiple screen sizes, which is a huge downfall.
